All-but-one of these people are lying (On hold)

aka. 'abootpal'. An online-multiplayer adaptation of the game 'Two Of These People Are Lying', as popularised by The Technical Difficulties. My first attempt at a multiplayer web game, built using Colyseus and Node.js.

This is only half-finished, and still needs some UI additions among other things. For now, you can start a game by typing '/game start' in the chat.
